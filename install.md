# Installation

## argparse
pip install argparse

## tqdm
pip install tqdm

## networkx
pip install networkx==2.2

### nltk
pip install -U nltk

### node2vec (and deepwalk)
pip install node2vec

### pywsd
pip install -U nltk
python -m nltk.downloader 'popular'
pip install -U pywsd

### Mangoes
pip install mangoes
